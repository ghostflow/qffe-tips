## 你可能需要的一些前端工具
	
* chrome插件
  * editThisCookie (查看修改设置cookie)
  * colorZilla （获取页面中rgb值）
  * modify headers （修改request header）
  * allow-control-allow-origin: * （修改response header）
* **sass**/less/postcss
* **grunt**/gulp/webpack
* **bower**/npm 3
* node
* charles
* sketch
* iterm2
* git
* netease cloud music (for better coding)
